# Unforgettable

치매로 기억이 흐릿해진 할머니의 기억을 손자가 되짚어가는 스토리 기반 퍼즐 어드벤처 게임입니다.  
플레이어는 현실 세계인 할머니의 방과 기억 속 세계를 오가며 퍼즐을 해결하고  
기억의 조각을 모아 할머니가 간직하고 있던 소중한 기억을 이해해가는 구조로 설계되었습니다.

기억 속 세계에서는 강아지 캐릭터로 플레이하며  
치매로 인해 왜곡된 공간, 감각, 방향 인지를 게임 플레이로 경험하게 됩니다.

플레이어는 3일 동안 기억 조각을 모으며 퍼즐을 해결하고  
현실 세계와 기억 세계를 오가며 할머니의 기억을 복원하게 됩니다.

---

## 개발 환경
- Engine: Unity
- Language: C#
- Platform: PC
- Genre: 스토리 기반 퍼즐 어드벤처
- Development: 팀 프로젝트 (5인)

---

## 담당 역할
- 플레이어 이동 및 카메라 추적 시스템 구현
- 기억 세계 탐색을 위한 노드 기반 경로 유도 시스템 구현
- 거리 기반 사운드 왜곡 및 감각 피드백 시스템 구현
- 발자국 기반 시각 피드백 시스템 구현
- 퍼즐 이벤트 진행 로직 구현 (Day1 / Day2 퍼즐 상호작용)
- 씬 기반 세이브 및 상태 복원 시스템 구현
- 오브젝트 상호작용 및 퍼즐 진행 상태 관리 시스템 구현

---

## 핵심 기능

### 플레이어 시스템
- 카메라 기준 이동 방향 계산
- Rigidbody 기반 캐릭터 이동
- 이동 방향에 따른 캐릭터 회전 보간
- 카메라 추적 및 시점 보정

관련 스크립트  
PlayerController  
CameraFollow  

---

### 노드 기반 경로 유도 시스템
- 기억 세계 탐색을 위한 NodePath 구조 설계
- 플레이어 위치와 경로 사이 거리 기반 상태 계산
- 경로 이탈 정도에 따른 시각 피드백 제공

관련 스크립트  
PathSystem  

---

### 감각 왜곡 시스템
- 경로 이탈 정도에 따라 사운드 왜곡 발생
- Pitch / Lowpass / Distortion / Flanger 효과 적용
- 방향 감각 혼란을 표현하는 입력 왜곡 시스템

관련 스크립트  
FootprintSpawner_forSound  
SoundNode  

---

### 발자국 시각 피드백 시스템
- 플레이어 이동 시 발자국 형태의 빛 생성
- 경로 상태에 따라 색상 변화
- 오브젝트 풀 기반 성능 최적화

관련 스크립트  
FootprintSpawner  

---

### 퍼즐 이벤트 시스템
- Day 단위 스토리 퍼즐 진행 구조
- 오브젝트 상호작용 기반 퍼즐 해결
- 대화 시스템과 연동된 이벤트 진행

관련 스크립트  
Day1Controller  
Day2Controller  

---

### 씬 기반 세이브 시스템
- 씬별 플레이어 위치 저장
- 이벤트 진행 상태 저장
- 제거된 오브젝트 상태 복원
- Memory World ↔ 현실 세계 위치 복원

관련 스크립트  
SaveManager  
GameSceneManager  

---

## 프로젝트 특징
- 치매로 인한 인지 왜곡을 게임 시스템으로 표현
- 현실 세계와 기억 세계를 오가는 구조 설계
- 노드 기반 경로 시스템을 통한 탐색 유도
- 사운드 및 입력 왜곡을 활용한 감각적 플레이 경험 구현
