# Branch Name Convention

- [Jira 티켓 넘버]_[기능명(소문자+하이픈 조합)]

  - ex) SMP-15_free-play

# Git Commit Message Convention
 

### Summary 

- [타입 (Added / Modified / Fixed / Deleted)] [내용]

  - 타입은 여러가지가 중복될 수 있음
    - Added : 씬 추가, 특정 오브젝트 및 스크립트 추가 등
    - Modified : 이전에 Added를 통해 올렸던 것을 수정
    - Fixed : Bug 등 Issue에 따른 수정
    - Deleted : 특정 오브젝트, 스크립트 삭제 등
  - 내용은 자유롭게 작성


### Description 

- 자유롭게 작성
